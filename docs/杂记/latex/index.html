LaTeX

用于排版的好东西，有点像高级 MarkDown

相关网站​

一些很好用的网站，用于快速入门 LaTeX

  • 知乎 LaTeX 教程1，教程2

  • Overleaf：在线 LaTeX 编辑器

  • 表格生成

文献 .bib​

在 Google Scholar 中可以复制 BibTeX 格式

@article{vaswani2017attention,
  title={Attention is all you need},
  author={Vaswani, Ashish and Shazeer, Noam and Parmar, Niki and Uszkoreit, Jakob and Jones, Llion and Gomez, Aidan N and Kaiser, {\L}ukasz and Polosukhin, Illia},
  journal={Advances in neural information processing systems},
  volume={30},
  year={2017}
}

使用时

\cite{vaswani2017attention}

\bibliographystyle{plain}  % 放在最后
\bibliography{ref}  % .bib文件名

文本 .tex​

对大多数命令来说， 为必填参数，[ ] 为可选参数。命令不需要闭合，环境需要闭合（\begin \end）

文档​

文件类型有英文也有中文，英文的可以通过宏包设置中文

\documentclass[12pt, a4paper, oneside]{ctexart} % 用于设置文件类型

\begin{document} % 正文必须放里面, 外面的不会出现在文件中

这里是正文. 

\end{document}

或者

\usepackage[UTF8]{ctex}

宏包​

像其他语言导入库一样，LaTeX 也有

\usepackage[bookmarks=true, linkcolor=black]{hyperref} % 宏包加载功能

导言​

导言可以设置很多信息

\title{}
\author{}
\date{\today}

\maketitle % 以在文档中设置标题

正文​

缩进由 LaTeX 控制，空行视作两行

段1
段1

段2
\newpage % 另起一页
\textbf{}\textit{} % 字体

字体颜色需要宏包 xcolor

\textcolor{red}{红色字}

章节​

一级、二级、三级标题

\section{一级}
\subsection{二级}
\subsubsection{三级}

四级标题需要宏包 enumitem

\begin{enumerate}[label=\alph*)] % a), b), c) ...
    \item xxx
\end{enumerate}

目录​

目录生成，需要生成两次，第一次生成 .toc 文件

\tableofcontents

图片​

需要使用宏包 graphicx

\begin{figure}s
    \includegraphics{图片.jpg}
    \caption{图片标题}
\end{figure}

表格​

直接生成喵

列表​

无序列表 itemize，有序列表 enumerate，描述 description

\begin{enumerate}
    \item[(1)] 这是第一点; 
    \item[(2)] 这是第二点;
    \item[(3)] 这是第三点. 
\end{enumerate}

定理环境​

需要使用宏包 amsthm

% 导言声明
\newtheorem{theorem}{定理}[section] % 编号依赖于 section (如 定理 1.1, 定理 1.2)
% 正文使用
\begin{theorem}[勾股定理] % [ ] 内是可选的定理名称
    在直角三角形中，斜边8的平方等于两直角边的平方和
\end{theorem}

页面​

需要使用宏包 geometry

\geometry{left=2.54cm, right=2.54cm, top=3.18cm, bottom=3.18cm} % 页边距
\linespread{1.5} % 行距

公式​

需要使用宏包 amsmath

$$ %行内
\[

\] % 行间

右侧有公式那种需要使用 equation 还要结合定理环境

\begin{equation}
E = mc^{2}
\end{equation}

代码​

代码，宏包 listings, xcolor

% 导言声明
\lstset{
    basicstyle=\ttfamily\small,
    keywordstyle=\color{blue},
    commentstyle=\color{purple},
    stringstyle=\color{red},
    numbers=left,
    numberstyle=\tiny\color{gray},
    frame=single,
    breaklines=true
}
% 正文使用
\begin{lstlisting}[language=Python]
# 这是一个 Python 示例
def hello_world():
    print("Hello,LaTeX!")
\end{lstlisting}

伪代码，宏包 algorithm, algpseudocode, amsmath

\begin{algorithm}
    \caption{欧几里得算法}\label{euclid}
    \begin{algorithmic}[1]  % [1] 显示行号
        \Procedure{Euclid}{$a,b$}
            \Comment{计算 a 和 b 的最大公约数}
            \State $r \gets a \bmod b$
            \While{$r \neq 0$}
                \Comment{当余数不为 0 时循环}
                \State $a \gets b$
            \EndWhile
            \State \textbf{return} $b$\Comment{最大公约数}
        \EndProcedure
    \end{algorithmic}
\end{algorithm}

假如只是文件目录，可以使用宏包 fontspec, fancyvrb

% 声明
\usepackage{fontspec, fancyvrb}
\setmonofont{DejaVu Sans Mono} % 支持 box-drawing 字符
% 使用
\begin{Verbatim}[frame=single, framesep=2mm, baselinestretch=0.9]
project/
├── src/
│   ├── main.py
│   └── utils.py
├── docs/
│   └── readme.md
└── config.yaml
\end{Verbatim}

引用​

对于公式、图片、表格引用方式不同，但大差不差

% 声明
\lable{eq:xxx} % 公式eq 图片fig 表格tab
% 调用
\eqref{eq:xxx} % 公式eqref 图片表格ref
