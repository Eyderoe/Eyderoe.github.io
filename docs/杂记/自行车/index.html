变速调整​

出现跳档或者定位不准的时候，就需要调整了。难绷，按视频装模做样调了三个下午，不如花20去车店调三分钟，难绷，下次调的时候再说

结构​

需要关注的大概主要是这一块

左边的两个

  • 飞轮（挡位越大齿片越小）、变速器（后拨）

右边的两个

  • 牙盘（挡位越大齿片越大）、变速器（前拨）、曲柄

前拨​

前拨的变速非常简单粗暴，链条穿过导链槽，钢丝线拉动导链槽左右移动实现换挡，一般不会出现什么大问题。出问题的时候说明问题已经比较大了

限位螺丝

  • L，限制

张力螺丝：用于调整定位

刹车维护​

刹车一般分为碟刹和圈刹，制动方式有线拉和油刹

单侧.线拉.碟刹​

结构如图：

  • 黄色：钢索，传递把手的拉力
  • 绿色：刹车片（来令片），耐磨摩擦块
  • 蓝色：弹簧片，释放刹车的时候撑开刹车片
  • 红色：碟片，减速 散热

当磨片厚度过小时需要更换来令片。当磨片厚度足够，但把手捏死才能制动，则需调整刹车间隙。两者的步骤大致相同

蓝色部分螺丝将刹车固定在车架上，取下后即可更换令片。令片更换比较简单，这里不过多赘述。主要是更换之后的刹车间隙调整，间隙过小会造成蹭碟，间隙过大则会让制动行程过大

单侧碟刹意味只有在刹车时只有一侧刹车片移动，另一侧刹车通过红色部分调节旋钮调整，这也是唯一调整刹车间隙大小的方式

将刹车装上后拧动螺丝至刹车晃动较小，调整红色部分至刹车间隙略大于碟片厚度即可，调整蓝色部分使碟片居中，拧紧蓝色部分。还有一种方法是先旋转红色，使安装时固定侧能贴住碟片，此时可以异常轻松的拧紧蓝色部分，之后再回拧红色部分，使存在缝隙，这种方式应该会更加简单？另外，刹车安装位置较小，尽量不要触摸碟片，碟片对油污过敏
