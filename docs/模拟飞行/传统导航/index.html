参数解释​

遵循 ARINC 424-18，有

  • fix (5.13): contains the five-character-name-code, or other series of characters with which the fix is identified.
  • turn (5.20): specifies the direction in which Terminal Procedure turns are to be made.
  • path (5.21): defines the path geometry for a single record of an ATC terminal procedure.
  • nav (5.23): allows the reference facility for the waypoint in a given record “Fix Ident” field.
  • theta (5.24): defined as the magnetic bearing to the waypoint identified in the record’s “FIX Ident” field from the Navaid in the “Recommended Navaid” field.
  • rho (5.25): defined as the geodesic distance in nautical miles to the waypoint identified in the record’s “Fix Ident” field from the NAVAID in the “Recommended NAVAID” field.
  • course (5.26): published outbound magnetic course from the waypoint identified in the record’s “Fix Ident” field.
  • distance (5.27): segment distances/along track distances/excursion distances/DME distances.
