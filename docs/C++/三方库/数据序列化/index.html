数据序列化

数据序列化的一些库

JSON​

经典数据序列化格式，使用 nlohmann-json。极致轻量化（仅单头文件）并在 Github 给出了一般例子

#include "json.hpp"
using json = nlohmann::json; // 一般约定

基本操作​

// 解析文本
json ex1 = json::parse(R"(
    {
    "pi": 3.141
    }
    )");
json ex2 = "{ \"happy\": true, \"pi\": 3.141 }"_json; // 用户定义字面量运算符
// 解析文件
std::ifstream inputFile;
json ex3 = json::parse(inputFile);
// 直接初始化
std::vector<int> list{1, 3, 5, 7};
json ex4{{"pi", 3.14}, {"array", list}};
// 导出至文本
std::string out = ex4.dump(4);
// 导出至文件
std::ofstream outputFile;
outputFile << ex4;

一般操作​

json ex{{"pi", 3.14}, {"array", {"abc", 123, nullptr, true}}};
// 查找值 兼容STL
auto it = ex.find("pi");
if (it != ex.end())
    double pi = it.value();
// 数组
auto &vec = ex["array"];
vec.push_back(54.8);

YAML​

可读性更高的一种序列化格式，这里使用 yaml-cpp 0.8 符合 YAML 1.2 标准，不过没 JSON 库那么好用

编译​

cmake 版本过高需调整 CMakeLists.txt

cmake_minimum_required(VERSION 3.5)

首先下载源码后创建进入 build 目录

cmake -G "MinGW Makefiles" ..
mingw32-make

创建进入 install 目录

cmake --install . --prefix "xxx/yaml-cpp-0.8.0/install/lib/cmake"

把 xxx/yaml-cpp-0.8.0/install 加入系统目录

在 CMakeList 中有

find_package(SQLiteCpp REQUIRED)

target_link_libraries(sql SQLiteCpp)

基本使用​

// 解析文本
YAML::Node ex1 = YAML::Load(R"(
    name: Example
    version: 1
    enabled: true
)");
// 解析文件
std::string inputFilePath;
YAML::Node ex2 = YAML::LoadFile(inputFilePath);
// 直接初始化
YAML::Node ex3;
ex3["name"] = std::vector<int>{1, 2, 3, 4, 5};
// 导出至文本
std::stringstream stream;
stream << ex3;
// 导出至文件
std::fstream outputFile;
outputFile << ex3;

一般操作​

YAML::Node node;
node["name"] = std::vector<int>{1, 2, 3, 4, 5};
// 查找值
if (auto exist = node["name"]; exist) {
    if (exist.IsSequence())
        std::vector<int> vec = exist.as<std::vector<int>>();
}
// 数组
auto vec = node["name"];
vec.push_back(5);
