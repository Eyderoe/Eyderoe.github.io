GeographicLib 地理计算​

在 Github 上提供了源码，在 SourceForge 上提供了安装包，文档 也比较详细

编译​

虽然提供了安装包，但是通过 MSVC 编译的 (.lib)，不能 MinGW 使用。需要自行编译，创建 build 目录后，运行

cmake -G "MinGW Makefiles" ..
mingw32-make

同样，将 bin 目录加入系统路径后，项目中 CMakeLists 写

cmake_minimum_required(VERSION 3.30)
project(geocpp)

set(CMAKE_CXX_STANDARD 17)

find_package(GeographicLib REQUIRED)
include_directories(${GeographicLib_INCLUDE_DIRS})

add_executable(geocpp main.cpp)

target_link_libraries(geocpp ${GeographicLib_LIBRARIES})

功能​

有如下约定

#include <GeographicLib/Geodesic.hpp>

using namespace GeographicLib;

struct Fix {
    double lat, lon;
};

距离 方位角​

求 b 相对于 a 的距离和方位角，距离单位：米，angle1 为 a 至 b 的初始方位，angle2 为 b 至 a 的初始方位

void fixRelation () {
    const Geodesic &geo = Geodesic::WGS84();
    Fix a{29.74667, 106.65334}, b{29.85167, 106.84666};
    double distance, angle1, angle2;
    geo.Inverse(a.lat, a.lon, b.lat, b.lon, distance, angle1, angle2);
}

PBD 点计算​

a 方位 90° 距离 3000 米处 b 的坐标

void pointBearingDistance () {
    const Geodesic &geo = Geodesic::WGS84();
    Fix a{29.74667, 106.65334}, b{};
    geo.Direct(a.lat, a.lon, 90, 3000, b.lat, b.lon);
}

磁差​

需要在 SourceForge 上或者 NOAA 上下载磁差模型，并建立目录 C:\ProgramData\GeographicLib\magnetic 放置文件

计算 a 处磁场信息，给了很多数据，但一般只会用到部分

#include <GeographicLib/MagneticModel.hpp>
#include <GeographicLib/Utility.hpp>

void magvar () {
    MagneticModel mag = MagneticModel(R"(wmm2025)");
    double t = Utility::fractionalyear<double>("2026-06-24");
    Fix a{29.74667, 106.65334};

    double x, y, z, xt, yt, zt; // 磁场分量和年变化率
    mag(t, a.lat, a.lon, 1600, x, y, z, xt, yt, zt);
    double H, F, D, I; // 水平强度, 总强度, 磁偏角, 磁倾角
    MagneticModel::FieldComponents(x, y, z, H, F, D, I);
}
