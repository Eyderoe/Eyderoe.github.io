约束

C++20 加入的强大功能，在不改变模板的情况下，通过 SM 调教的更通人性

类型约束​

核心思想：从之前的一坨模板，变为编译时可以检查的东西

通过 requires 可以启用子句，这里 std::totally_ordered<T> 是一个概念要求能进行比较

// 要求类型可比较
template <typename T>
    requires std::totally_ordered<T>
T max (T a, T b) {
    return a > b ? a : b;
}

还能定义 requires 表达式，这里要求 T 能输出到 cout

// 要求类型可输出
template <typename T>
    requires requires(T a) {
        std::cout << a << std::endl;
    }
T max (T a, T b) {
    return std::cout << (a > b ? a : b) << std::endl;
}

当然也能通过 && 和 || 进行拼接
