1 绪论

引入飞行程序设计中一些重要概念

1.1 概念​

航空器在终端区内起飞或下降着陆时的路线

1.1.1 准则​

遵循：ICAO 8168 航空器运行、MH/T 4023-2007 目视和仪表飞行程序设计规范

1.2 参数​

1.2.1 航空器速度​

VatV_{at}Vat​ 定义为 max(1.23Vslg,1.3Vso)max(1.23V_{slg},1.3V_{so})max(1.23Vslg​,1.3Vso​)，并根据 VatV_{at}Vat​ 将航空器分类为 A-E 以及 H 直升机。不同分类航空器在设计飞行程序时各个阶段所用的指示空速不同，设计时还要将指示空速转换为真空速

R=562tanαVR=\frac{562tan\alpha}{V}R=V562tanα​

等待和进近时平均坡度为25°，目视盘旋20°，离场和复飞15°，转弯率小于(3°/s)。无风时转弯半径为：

1.2.2 风​

风速和风向都在不断变化，故使用全向风来代替特定风向，航空器转过一定角度形成的轨迹为风螺旋线。同样的，设计时各个阶段所用全向风不同

1.3 定位点及定位容差​

定位点附带相关航段信息，其定位精度对程序安全性和可靠性有直接影响

1.3.1 导航容差​

定位点可以采用：飞越导航台、双台定位、雷达定位三种方式

定位容差区：由于地面和机载设备精度限制、驾驶员技术误差，航空器定位时可能产生的偏差范围。沿标称航迹的长度为定位容差，进入容差区的长度一般记为 d1d_1d1​，飞出为 d2d_2d2​

  1. 飞越导航台

    • VOR：50° 倒圆锥区，进入误差为 ±5°，保持误差为 ±5°
    • NDB：40° 倒圆锥区，进入误差为 ±15°，保持误差为 ±5°
    • 指点标：抛物线

  2. 交叉定位点

    对于提供航迹引导的导航台为三者平方和根，侧方定位的导航台不考虑飞行技术容差。需要注意的是交叉定位点对导航台到定位点夹角有限制

    • VOR：地面系统、接收机、飞行技术
    • NDB：地面设备、机载设备、飞行技术
    • LOC：地面监测、机载设备、飞行技术

    DME的测距精度随距离变化，为：±[0.25NM+1.25%Distance]±[0.25NM+1.25\%Distance]±[0.25NM+1.25%Distance]

1.4 超障保护区和超障裕度​

1.4.1 超障保护区​

保护区的保障不是绝对的，而是 10−710^{-7}10−7 碰撞风险水平，通常根据概率可容度（保护空域可容纳沿航路飞行的总飞行时间百分比）确定范围。导航容差 99.7% 概率可容度确定超障保护区，按 95% 确定主区，其余区域为副区
