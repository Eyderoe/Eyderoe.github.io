Transformer

谷歌于 2017 发表的革命性论文 Attention Is All You Need，奠定了至今几乎所有大语言模型的基础

前置工作​

Transformer 最原始工作是语句翻译

RNN​

序列模型，序列中每个词都会使用先前的状态 ht−1h_{t-1}ht−1​ 和当前词计算 hth_tht​，来实现保留过去状态。但存在并行化较为困难，序列较长也会造成 hth_tht​ 较大

Attention​

注意力机制在 Transformer 前就提出了（主要两种：加型、点乘），但都是用于 RNN 的结合。Transformer 主要的贡献为自注意力、多头注意力和纯注意力模型

编解码​

编码器和解码器，解码器能对长 n 的序列能表示为每个词转换为向量的序列 (x_1,x_2,...,x_n)，解码器通过编码器输出生成一个序列 (y_1,y_2,...,y_n)。编码器能一次看到整个句子，但解码器只能单词输出（自回归），过去的输出作为当前的输入，有点像 RNN 但 RNN 会压缩信息，自回归看的信息是完整的

embedding 层是把输入的词转换为向量

模型架构​

编解码器​

编码器 6 个重复的子块组成，子块有一个多头注意力和一个简单的 MLP。每个输出维度 dmodel=512d_{model}=512dmodel​=512

解码器多了一个掩码，来避免看见未输出的语句

归一化​

对二维来说 batch-norm 会把每一个特征的均值变为 0 方差为 1，layer-norm 是对样本做的操作。layer 只在自己里面算不需要全局，一般用的更多
