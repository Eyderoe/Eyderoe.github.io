基本概念​

贯穿整个课程的定义，基石

格子世界 Grid-World​

智能体在一个格子划分的环境中运行

状态 State​

智能体相对于环境的一个状态

所有状态组成了状态空间，在格子世界中 S={si}i=19S=\{s_i\}_{i=1}^9S={si​}i=19​

行为 Action​

在一个状态，智能体能做的行为

同样的，也有行为空间，在格子世界中 A(si)={ai}i=15A(s_i)=\{a_i\}_{i=1}^5A(si​)={ai​}i=15​，原地不动和上下左右四个方向移动

状态通过行为进行转移，s1→a2s2s_1\overset{a_2}{\rightarrow} s_2s1​→a2​s2​。状态转移定义了智能体与环境的交互，可以用条件概率描述 p(s2∣s1,a2)=1p(s_2|s_1,a_2)=1p(s2​∣s1​,a2​)=1，即 s1s_1s1​ 采取 a2a_2a2​ 一定会转移到 s2s_2s2​

策略 Policy​

智能体在某个状态会选择什么样的行为

也可以通过概率描述，有 π(a2∣s1)=1\pi(a_2|s_1)=1π(a2​∣s1​)=1 即在 s1s_1s1​ 时一定会采取 a2a_2a2​

奖励 Reward​

强化学习中独特的概念，定义了智能体在采取行动后得到的东西

出边界可以为 rbound=−1r_{bound}=-1rbound​=−1，奖励只是一个相对量，因为智能体最终的目的是惩罚最少或者说奖励最多。也能写成条件概率形式 p(r=−1∣s1,a1)p(r=-1|s_1,a_1)p(r=−1∣s1​,a1​)

轨迹 Trajectory​

轨迹是 状态-行为-奖励 的链条，就像 s1→r=0a2s2→r=0a2s3s_1\xrightarrow[r=0]{a_2}s_2\xrightarrow[r=0]{a_2}s_3s1​a2​r=0​s2​a2​r=0​s3​

回报 Return​

和奖励有点像，奖励是对每一个行为的。回报是对每一条轨迹而言的，一坨轨迹所有奖励总和，可以用于刻画一条轨迹的良好程度，通常用 G(t)G(t)G(t) 表示

但假如到达目标后没有结束，那轨迹就会无限长，回报发散到无限大。现在引入折扣率 Discounted Reate γ∈[0,1)\gamma \in [0,1)γ∈[0,1)，和折扣回报 r0γ0+r1γ1+...+rnγnr_0\gamma ^0+r_1\gamma ^1+...+r_n\gamma ^nr0​γ0+r1​γ1+...+rn​γn，不仅可以使回报收敛，还能调整近期和远期权重

回合 Episode​

假如智能体最终终止在一个状态，那么这样形成的轨迹就叫回合。相反的存在一些任务，智能体永不终止，也可以把可终止的换成持续性的

  1. 终止状态是特殊状态，在之后无回报
  2. 终止状态无所谓的，智能体能离开并再次进入

轮 Step​

一个回合内的单次动作过程，即智能体在某状态进行某动作，环境做出反馈并更新状态的过程

马尔可夫决策过程 Markov Decision Process (MDP)​

大框架，有几个关键元素

集合 Set

  • 状态：状态集合 SSS
  • 行为：行为集合 A(s),s∈SA(s),s \in SA(s),s∈S
  • 奖励：奖励集合 R(s,a)R(s,a)R(s,a)

概率 Probability Distribution

  • 状态转移概率：采取行为转移到其他状态概率 p(s′∣s,a)p(s'|s,a)p(s′∣s,a)
  • 奖励概率：获得某个奖励概率 p(r∣s,a)p(r|s,a)p(r∣s,a)

策略 Policy：选择行为的概率 π(a∣s)\pi(a|s)π(a∣s)

马尔可夫性质 Markov Property：未来只取决于现在而不是过去

蒙特卡洛 MC Basic​

model-based 下已知 p(s′,r∣s,a)p(s',r|s,a)p(s′,r∣s,a)，可进一步展开 qπ(s,a)q_\pi(s,a)qπ​(s,a)。model-free 就只能用原始的 qπ(s,a)=E[Gt∣St=s,At=a]q_\pi(s,a)=\mathbb{E}[G_t|S_t=s,A_t=a]qπ​(s,a)=E[Gt​∣St​=s,At​=a]，这样就能把上面的 model-based 的策略迭代方法通过估计给转换为 model-free

蒙特卡洛 MC Exploring Start​

和名字一样，每回合开始的时候会随机选择一个状态和动作开始，通过足够大的回合数来确保收敛到真实值

s1→a2s2→a4s1→a2s2→a3s5s_1\xrightarrow{a_2}s_2\xrightarrow{a_4}s_1\xrightarrow{a_2}s_2\xrightarrow{a_3}s_5s1​a2​​s2​a4​​s1​a2​​s2​a3​​s5​

对于上面这个回合，有下面两种计算回报策略

  • first-visit：只用第一次遇到的未来回报

  • every-visit：用每一次遇到的未来回报

并且不需要等到所有回合完才更新，每次回合后就直接更新行动价值，并且更新这个回合中涉及到的每对状态动作

广义策略迭代 Generalized Policy Iteration​

先插入这个概念，即强大通用的学习框架，绝大部分强化学习都能看见这个影子。他把单步分成了两块：

  1. 策略评估：观察当前策略表现，评估每个状态/动作有多好
  2. 策略改进：根据评估调整策略，每个状态选择更好的动作

评估和改进循环往复，相互驱动

ϵ→1\epsilon\rightarrow 1ϵ→1 表示均匀或者说随机选择动作，反之更倾向于上面的 Exploring Start

相比于 Exploring Start， 为选择添加一点随机性避免了局部最优，稍微的随机保证了持续探索。但也正因此，这也不能收敛到最优（最优策略下，仍会以ε的概率进行随机探索）

随机近似与随机梯度下降​

第一次学可能会感觉反直觉的东西，并伴随着更多的数学公式。和前面关系不太大，对后面打基础

罗宾斯 - 门罗 Robbins-Monro algorithm​

求解 g(w)=0g(w)=0g(w)=0 通常可以通过 g(w)=∇wJ(w)=0g(w)=\nabla_w J(w)=0g(w)=∇w​J(w)=0 来求解（求函数根转换为求损失函数最小，最小的必要条件为梯度为零）。然后很神奇的东西来了，即使不知道 g(w)g(w)g(w) 表达式，也可以求。反正模型和数据总要有一个

wk+1=wk−akg~(wk,ηk)w_{k+1} = w_k - a_k \tilde{g}(w_k, \eta_k)wk+1​=wk​−ak​g~​(wk​,ηk​)

  • wkw_kwk​ 是参数估计值
  • aka_kak​ 是学习率
  • g~\tilde{g}g~​ 是带噪声的观测值，ηk\eta_kηk​ 是均值为 0 的噪音

这里忽略一大坨数学证明，然后有（学习率足够大保证最优，足够小保证收敛）

随机梯度下降 Stochastic gradient descent​

其实可以从上面一直看下来会发现，一直在用估计来代替真实值（蒙特卡洛和罗宾斯门罗）

再把上面的表达式改写下：J(w)=E[f(w,X)]J(w) = \mathbb{E}[f(w, X)]J(w)=E[f(w,X)]

梯度下降 gradient descent GD

wk+1=wk−αk∇wE[f(wk,X)]=wk−αkE[∇wf(wk,X)]w_{k+1} = w_k - \alpha_k \nabla_w \mathbb{E}[f(w_k, X)] = w_k - \alpha_k \mathbb{E}[\nabla_w f(w_k, X)]wk+1​=wk​−αk​∇w​E[f(wk​,X)]=wk​−αk​E[∇w​f(wk​,X)]

这里用了一个很高级的东西，梯度的期望等于期望的梯度。但 E[∇wf(wk,X)]\mathbb{E}[\nabla_w f(w_k, X)]E[∇w​f(wk​,X)] 这个也不知道

批量梯度下降 batch gradient descent BGD

有数据的话就可以用这种方法

E[∇wf(wk,X)]≈1n∑i=1n∇wf(wk,xi)\mathbb{E}[\nabla_w f(w_k, X)] \approx \frac{1}{n} \sum_{i=1}^{n} \nabla_w f(w_k, x_i)E[∇w​f(wk​,X)]≈n1​∑i=1n​∇w​f(wk​,xi​)

效率太低下了，每次更新就要一大坨计算

随机梯度下降 Stochastic gradient descent SGD

很简洁的公式，估计的次数少了但相应精度也会降低，需要通过调整 αk\alpha_kαk​ 来保证收敛（炼丹的由来）

E[∇wf(wk,X)]≈∇wf(wk,xk)\mathbb{E}[\nabla_w f(w_k, X)]\approx \nabla_w f(w_k, x_k)E[∇w​f(wk​,X)]≈∇w​f(wk​,xk​)

又经过一坨证明，可以知道随机梯度下降就是一种特殊的 RM 算法

最小批量梯度下降 minimal batch gradient descent MBGD

E[∇wf(wk,X)]≈1m∑i∈Tk∇wf(wk,xj)\mathbb{E}[\nabla_w f(w_k, X)]\approx \frac{1}{m} \sum_{i \in \mathcal{T}_k} \nabla_w f(w_k, x_j)E[∇w​f(wk​,X)]≈m1​∑i∈Tk​​∇w​f(wk​,xj​)

这个是在采样中抽签。当 m→1m\rightarrow 1m→1 就会更像 SGD，m→nm\rightarrow nm→n 就会更像 BGD

时序差分方法​

这也是 model-free 方法

回顾​

w=E[X]w = \mathbb{E}[X]w=E[X] 通过改写为 g(w)=w−E[X]=0g(w) = w - \mathbb{E}[X]=0g(w)=w−E[X]=0 可以用 RM 算法求解

w=E[v(X)]w = \mathbb{E}[v(X)]w=E[v(X)] 也可以通过改写然后用 RM 求解

w=E[R+γv(X)]w = \mathbb{E}[R + \gamma v(X)]w=E[R+γv(X)] 这个虽然看着更麻烦，但也可以拆开来求解

时序差分 Temporal Difference​

结合了蒙特卡洛和动态规划，可以从不完整的经验中学习：

vt+1(st)=vt(st)−αt(st)[vt(st)−(rt+1+γvt(st+1))]v_{t+1}(s_t) = v_t(s_t) - \alpha_t(s_t) \left[ v_t(s_t) - \left( r_{t+1} + \gamma v_t(s_{t+1}) \right) \right]vt+1​(st​)=vt​(st​)−αt​(st​)[vt​(st​)−(rt+1​+γvt​(st+1​))]

vt+1(s)=vt(s),∀s≠stv_{t+1}(s) = v_t(s), \quad \forall s \neq s_tvt+1​(s)=vt​(s),∀s=st​

然后引申出：

  • 目标，对当前状态价值更优的估计：vˉt≐rt+1+γv(st+1)\bar{v}_t \doteq r_{t+1} + \gamma v(s_{t+1})vˉt​≐rt+1​+γv(st+1​)
  • 误差，与更优目标间的差值（带噪声的观测）：δt≐v(st)−[rt+1+γv(st+1)]=v(st)−vˉt\delta_t \doteq v(s_t) - \left[ r_{t+1} + \gamma v(s_{t+1}) \right] = v(s_t) - \bar{v}_tδt​≐v(st​)−[rt+1​+γv(st+1​)]=v(st​)−vˉt​

和蒙特卡洛不一样，蒙特卡洛需要完整的回合结束后才能更新，时序差分每走一轮就可以更新。简单的说就是用误差和学习率来更新值

时序差分 Sarsa​

改为了行动价值，策略是 ε-Greedy

qt+1(st,at)=qt(st,at)−αt(st,at)[qt(st,at)−(rt+1+γqt(st+1,at+1))]q_{t+1}(s_t, a_t) = q_t(s_t, a_t) - \alpha_t(s_t, a_t) \left[ q_t(s_t, a_t) - \left( r_{t+1} + \gamma q_t(s_{t+1}, a_{t+1}) \right) \right]qt+1​(st​,at​)=qt​(st​,at​)−αt​(st​,at​)[qt​(st​,at​)−(rt+1​+γqt​(st+1​,at+1​))]

qt+1(s,a)=qt(s,a),∀(s,a)≠(st,at)q_{t+1}(s, a) = q_t(s, a), \quad \forall (s, a) \neq (s_t, a_t)qt+1​(s,a)=qt​(s,a),∀(s,a)=(st​,at​)

时序差分 Expected Sarsa​

把 γqt(st+1,at+1)\gamma q_t(s_{t+1}, a_{t+1})γqt​(st+1​,at+1​) 换成了 γE[qt(st+1,A] \gamma\mathbb{E[q_t(s_{t+1}, A ]}γE[qt​(st+1​,A]

因为引入了期望，计算量增大了，换来了更快的收敛

时序差分 n-step Sarsa​

多继承自蒙特卡洛和 Sarsa，又是听不懂的一坨

时序差分 Q-learning​

和基本的时序差分估计状态价值、Sarsa 估计行动价值不同（实际采取的动作），直接估计了最优行动价值（下一个状态中估计的最优动作）

qt+1(st,at)=qt(st,at)−αt(st,at)[qt(st,at)−[rt+1+γmax⁡a∈Aqt(st+1,a)]]q_{t+1}(s_t, a_t) = q_t(s_t, a_t) - \alpha_t(s_t, a_t) \left[ q_t(s_t, a_t) - \left[ r_{t+1} + \gamma \max_{a \in \mathcal{A}} q_t(s_{t+1}, a) \right] \right]qt+1​(st​,at​)=qt​(st​,at​)−αt​(st​,at​)[qt​(st​,at​)−[rt+1​+γmaxa∈A​qt​(st+1​,a)]]

qt+1(s,a)=qt(s,a),∀(s,a)≠(st,at)q_{t+1}(s, a ) = q_t(s, a), \quad \forall (s, a) \neq (s_t, a_t)qt+1​(s,a)=qt​(s,a),∀(s,a)=(st​,at​)

细化策略：

  • 目标策略 Target：最优策略
  • 行为策略 Behavior：实际执行，用于生成数据的策略，一般包含随机性
  • 同策略 On：目标策略和行为策略是同一个
  • 异策略 Off：目标策略和行为策略不是同一个

Q-learning 就是异策略 ，比如行为策略 ε-Greedy 目标策略纯 Greedy。而且因为异策略，使 Q-learning 能和神经网络结合，而不是其他算法

值函数近似​

和上一节一样是 TD，但时序差分是基于表格（直观且易于分析），值函数近似是基于函数（拟合表格）。也是从这里开始引入神经网络

原理​

表格法再大一点可能就一坨了，引入函数来近似表格。好处是更少的内存占用，坏处是近似会造成精度损失，也可以用高阶或神经网络来做非线性拟合。以线性为例，ϕ(s)\phi(s)ϕ(s) 是特征向量（非线性的东西放这，对参数来说线性）

v^(s,w)=as+b=[s,1][ab]=ϕT(s)w\hat{v}(s, w) = as + b = [s, 1] \begin{bmatrix} a \\ b \end{bmatrix} = \phi^T(s) wv^(s,w)=as+b=[s,1][ab​]=ϕT(s)w

目标是找到参数 www 使估计值更接近真实值，比如常见的目标函数是均方差

J(w)=E[(vπ(S)−v^(S,w))2]J(w) = \mathbb{E}\left[ \left( v_\pi(S) - \hat{v}(S, w) \right)^2 \right]J(w)=E[(vπ​(S)−v^(S,w))2]

目标函数的期望是假设在稳态分布中的，关于稳态分布在前置知识那篇笔记里面有

但 vπ(st)v_\pi(s_t)vπ​(st​) 显然不能得到，可以用之前学过的方法近似得到

  • 蒙特卡洛：wt+1=wt+αt(gt−v^(st,wt))∇wv^(st,wt)w_{t+1} = w_t + \alpha_t \left( g_t - \hat{v}(s_t, w_t) \right) \nabla_w \hat{v}(s_t, w_t)wt+1​=wt​+αt​(gt​−v^(st​,wt​))∇w​v^(st​,wt​)
  • 时序差分：wt+1=wt+αt[rt+1+γv^(st+1,wt)−v^(st,wt)]∇wv^(st,wt)w_{t+1} = w_t + \alpha_t \left[ r_{t+1} + \gamma \hat{v}(s_{t+1}, w_t) - \hat{v}(s_t, w_t) \right] \nabla_w \hat{v}(s_t, w_t)wt+1​=wt​+αt​[rt+1​+γv^(st+1​,wt​)−v^(st​,wt​)]∇w​v^(st​,wt​)

v^(s,w)\hat{v}(s, w)v^(s,w) 选取也很多样
